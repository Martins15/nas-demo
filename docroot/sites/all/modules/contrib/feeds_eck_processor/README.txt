
"Import ECK entities using Feeds"

Features
========

- Entity Construction Kit processor for Feeds that creates ECK entities in Drupal.

Help using Feeds
================

- The "site builder's guide to Feeds" http://drupal.org/node/622698

Installation
============

- Install & enable the Feeds (including UI) and ECK module
- Go to admin/structure/feeds and add a new importer
- Select the "ECK processor"
- In Settings, select an entity type and bundle to use to create new ECK entities
